Feature extraction

There are a number of reasons you would want to detect some object within an image, or frames within a video. In this presentation, we will look at how to detect the pupil within an image of the eye. First we will explore how to detect edges in an image. Then we will explore how to detect a circle within that edge map. We will use this cleaned up image of my eye as our example throughout.

Sobel Filter (example of Edge Detection)

Previously we have explored discretization of differential equations to approximate differentiation. Consider an image in terms of its pixel values as a density function. Theses pixels change over the 2D space of the image. Thus there is a gradient function that we can approximate. For a derivation of this discretization method, see the references provided.

Calcuate the magnitude of the gradient
Then we may threshold the gradient magnitude to specify which point gradients represent edges. How to threshold can become a fairly compicated, so I define a simple threshold, T, as proportional to the average gradient magnitude:
input_img = imread('pupil1_fixed.png');
gray = rgb2gray(input_img);
A = double(gray);
% Sobel operation
Gx = [-1,0,1;-2,0,2;-1,0,1];
Gy = [-1,-2,-1;0,0,0;1,2,1];
% declare matrices for info storage
[rows, cols] = size(A);
mag = zeros(size(A));
% convolve Gx and Gy over A
for ii = 1:rows-2
for jj = 1:cols-2
Sx = sum(Gx.*A(ii:ii+2,jj:jj+2),'all');
Sy = sum(Gy.*A(ii:ii+2,jj:jj+2),'all');
mag(ii+1,jj+1) = sqrt(Sx.^2 + Sy.^2);
end
end
% Thresholding
% declare threshold - differs by application
threshold = mean(mag,'all');
threshold = threshold*3.2;
% enforce threshold
edge_map = zeros(size(mag));
edge_map(mag > threshold) = 1;
% plotting
figure;
subplot(1,2,1)
imshow(gray)
title('Grayscale Image')
subplot(1,2,2)
imshow(edge_map)
title('Edges of Image')

Circle Hough Transform

Suppose you have a perfect (decided by application) edge map of an image. If you want to find particular objects/structures within that map, you must infer them. The Circle Hough Transform allows you to collect candidate circles and then select the best candidate(s) based on local or global maxima within that collection.

min_radius = 10;
max_radius = 60;
[rows, cols] = size(edge_map);
% declare matrix for tracking votes for the Circle information
votes = zeros(rows,cols,max_radius);
% voting
for ii = 1:rows
for jj = 1:cols
if edge_map(ii,jj) == 1
% sweep through possible radii
for r = min_radius:max_radius
% sweep through phi degrees
for degrees = 1:360
% generate coordinates of circle about (jj,ii)
x = jj - int32(r*cos(degrees*pi/180));
y = ii - int32(r*sin(degrees*pi/180));
if (x+r >= cols) || (x-r < 0) || (y+r >= rows) || (y-r < 0)
continue;
end
if (x < cols) && (x > 0) && (y < rows) && (y > 0)
votes(y,x,r) = votes(y,x,r) + 1;
end
end
end
end
end
end
% get location of best circle
Y=0;X=0;R=0;V=0;
for ii = 1:rows
for jj = 1:cols
for kk = min_radius:max_radius
if votes(ii,jj,kk) > V
V = votes(ii,jj,kk);
Y = ii;
X = jj;
R = kk;
end
end
end
end
% plot circle on original image
theta = 0:pi/50:2*pi;
y = R*sin(theta)+Y;
x = R*cos(theta)+X;
figure;
subplot(2,2,1)
imshow(input_img)
title('Input image')
subplot(2,2,2)
imshow(gray)
title('Grayscale Image')
subplot(2,2,3)
imshow(edge_map)
title('Edges of Image')
subplot(2,2,4)
imshow(input_img);hold on;
plot(x,y,'r','LineWidth',3);
hold off;
title('Image with Circle Annotation')
